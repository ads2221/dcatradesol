### 🎯 Фильтры

#### Описание

Система фильтров для автоматического отбора токенов. Определяет, какие токены бот будет торговать.

#### Описание

Тонкая настройка параметров для отбора токенов и управления рисками.

#### Основные фильтры (редактируемые)

**💰 Мин. общая сумма DCA**

```
Нажмите кнопку → введите сумму в USD (например: 500000)
Эффект: Бот будет торговать только токены с хотя бы такой суммой DCA
```

**💎 Мин. MCAP / Макс. MCAP**

```
Нажмите кнопку → введите значение
Диапазон капитализации для торговли
Пример: Min 5000000, Max 300000000
```

**💧 Ликвидность**

```
Нажмите → введите минимальную ликвидность в USD
Чем выше ликвидность, тем меньше проскальзывание
Рекомендуется: $150,000+
```

**📊 Min Frequency**

```
Нажмите → введите число
Минимальное количество DCA ордеров для этого токена
Показатель популярности/интереса
Рекомендуется: 300-500
```

**⏱️ Min ETA / Max ETA**

```
Нажмите → введите минуты
Диапазон времени запуска токена
Пример: Min 5, Max 120 (означает 5-120 минут с рождения)
```

**📈 Potential Price Change (Min & Max)**

```
Нажмите → введите процент
Диапазон потенциала изменения ценыпара
Пример: Min 0%, Max 500%
```

**👥 Holders (Min & Max)**

```
Нажмите → введите число
Диапазон количества держателей токена
Защита от скамов (ищите много держателей)
```

**📊 Vol 24h (Min & Max)**

```
Нажмите → введите сумму в USD
Диапазон объема торговли за 24 часа
Показатель ликвидности и интереса
```

#### Умные правила

**⚙️ MCAP-Frequency правила**

```
Что это: Специальные правила на основе MCAP

Кнопки:
├── ➕ Добавить правило
├── ✅ Активное правило (нажмите для деактивации)
├── ✏️ Изменить
└── 🗑️ Удалить

Пример:
"Если MCAP < $5M, то требовать Min Frequency >= 1000"
```

**🪙 Токен-специфичные правила**

```
Что это: Правила для конкретных токенов

Кнопки:
├── ➕ Добавить правило
├── ✅ Активное правило
├── ✏️ Изменить
└── 🗑️ Удалить

Что можно сделать:
- Пропустить торговлю этого токена
- Изменить Min Frequency для него
- Изменить Min MCAP для него
```

**📉 Мин. PPC для токенов**

```
Что это: Минимальный потенциал цены для конкретных токенов

Пример:
Токен ABC → минимум 50% потенциала
Токен XYZ → минимум 100% потенциала
```

**🚫 Автоблокировка пользователей**

```
Что это: Автоматический черный список плохих трейдеров

Как работает:
1. Бот анализирует PnL пользователя
2. Если ниже порога - добавляет в черный список
3. Больше не торгует его позиции

Параметры:
├── Минимальное PnL для блокировки
├── Минимальное количество убыточных сделок
└── Период анализа
```

**💰 Правила сумм по фильтрам**

```
Что это: Динамическое изменение суммы торговли по параметрам

Пример:
├── Если MCAP > $50M → сумма = $50
├── Если MCAP < $10M → сумма = $10
└── Иначе → сумма = $20
```

#### Черный список

**🚫 Черный список токенов**

```
Кнопки:
├── ➕ Добавить один токен
├── 📋 Массовое добавление (много за раз)
├── 🗑️ Удалить токен
└── 📊 Просмотр списка (X токенов)

Как добавить:
1. Нажмите ➕ или 📋
2. Введите адрес токена(ов)
3. Подтвердите
```

**🚫 Черный список пользователей**

```
Кнопки: Как у черного списка токенов

Как добавить:
1. Нажмите ➕ или 📋
2. Введите username пользователя(ей)
3. Подтвердите

Когда использовать:
- Если пользователь часто входит в убыточные сделки
- Если его стратегия вам не нравится
- Чтобы избежать его позиций
```
