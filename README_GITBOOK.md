## 🚀 Как использовать

### Вариант 1: Импортировать в GitBook (Рекомендуется)

1. **Создайте GitHub репозиторий**
   - Создайте новый репозиторий `dca-bot-guide`
   - Инициализируйте Git

2. **Загрузите файлы**
   ```bash
   git clone <your-repo>
   cd dca-bot-guide
   cp -r project/guide/* .
   git add .
   git commit -m "Initial DCA Bot guide"
   git push origin main
   ```

3. **Подключите к GitBook**
   - Откройте https://www.gitbook.com/
   - Создайте новое пространство
   - Выберите "Git Sync"
   - Авторизуйте GitHub
   - Выберите ваш репозиторий
   - Синхронизируйте
   - Готово! 🎉

4. **Результат**
   - Профессиональная документация
   - Удобная навигация
   - Поиск по содержимому
   - Автоматическое обновление при push
   - Публичный доступ для пользователей

### Вариант 2: Использовать один файл

Если не хотите использовать Git Sync:
1. Скопируйте текст из `GITBOOK_GUIDE.md`
2. Создайте новый документ в GitBook
3. Вставьте текст
4. GitBook разбивает его на страницы

### Вариант 3: Использовать JSON для парсинга

Если нужна автоматизация:
1. Парсьте `BOT_BUTTONS_STRUCTURE.json`
2. Генерируйте документацию программно
3. Используйте данные для UI компонентов
